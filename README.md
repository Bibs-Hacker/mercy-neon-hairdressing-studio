## ⚡ Features

* 🌟 **Vibrant Neon Aesthetic:** A stunning visual theme utilizing modern glowing borders, custom glassmorphism, and custom variable-driven color schemes.
* 📱 **Flawless Fluid Responsiveness:** Built using high-performance utility wrappers (`relative min-h-screen`) ensuring perfect scaling across ultra-wide monitors, tablets, and smartphones.
* ⚡ **Ultra-Optimized Elements:** Inspected and fully debugged layout tree to deliver zero rendering blocks, minimal DOM depth, and lightning-fast paint metrics.
* ✂️ **Interactive Studio Catalog:** Clear, clean structural presentation designed to convert casual web visitors into scheduled salon clients.

## 📥 Installation & Local Setup

Get the studio environment running locally on your personal machine in less than two minutes:

```bash

# 2. Enter the codebase workspace

# 3. Launch the application inside your browser
open index.html
```
